# 视觉系统

## 设计目标

这套视觉服务 4-6 年级课堂展示，要做到：

- 能投影读清。
- 不像纯文字讲义。
- 有中小学生能接受的轻插画感。
- 不过度幼稚，不用杂乱卡通。
- 图标和插画风格统一。

## 页面类型

### 封面页

用途：建立主题。

视觉建议：

- 保持主题图形：真实问题、用户场景、AI 追问、项目作品。
- 可以加入统一线性图标，但不堆太多小元素。

### 大字号提问页

用途：老师单独提问，学生讨论。

规则：

- 深色背景。
- 一个大问题。
- 一句提示语。
- 3-4 个关键词。
- 右侧一个统一线性插画。

当前样式：

- 背景：深蓝 `#0f2f56`
- 大标题：58px
- 提示语：25px
- 插画：白色线性图标，圆形半透明底

### 内容页

用途：讲解知识、方法、判断标准。

规则：

- 可以使用 2-6 张卡片。
- 每张卡片尽量不超过 2 句。
- 重要卡片使用图标。

### 任务页

用途：学生当堂做事。

规则：

- 必须有“学生要做什么”。
- 必须有“留下结果”。
- 尽量给模板。

### 项目方向页

用途：四个项目识别和分组推进。

统一图标：

- 冰箱食材检测：冰箱图标。
- 果园果实病状检查：果实图标。
- 办公区无效照明监控：台灯图标。
- 车辆盲区视觉提醒：车辆图标。

## 颜色

基础色：

- 深蓝：`#123f72`
- 辅助蓝：`#1f6db1`
- 绿色：`#2f7d64`
- 橙色：`#b56a14`
- 红色：`#9d3f46`
- 背景：`#f5f7fb`
- 线框：`#d9e2ec`

柔和底色：

- 蓝：`#e8f2fb`
- 绿：`#e8f5ef`
- 橙：`#fff3df`
- 红：`#fae9ea`

## 图标系统

当前使用内嵌 SVG symbol，统一线性风格：

- `icon-search`
- `icon-lightbulb`
- `icon-user`
- `icon-map`
- `icon-check`
- `icon-ai`
- `icon-log`
- `icon-fridge`
- `icon-apple`
- `icon-lamp`
- `icon-car`
- `icon-question`
- `icon-mic`

后续新增图标要求：

- 线条宽度一致。
- 不使用填充复杂插画。
- 不使用多套图标库混搭。
- 图标只辅助识别，不承担正文信息。

## 不建议

- 不要每一页都做满卡片。
- 不要把大问题页再塞成内容页。
- 不要使用多套插画风格。
- 不要让插画遮挡标题。
- 不要使用太多渐变、装饰圆点和无意义背景图案。
