# ICC 第一次课模块化课件

这个文件夹把原来的 `ICC第一次课课件.html` 拆成可维护的课件模块。

## 维护原则

- `content/` 只写课程内容：讲什么、问什么、学生做什么、产出什么。
- `structure/` 只写页面顺序和页面类型：封面页、大问题页、任务页、案例页等。
- `visual/` 只写视觉系统：颜色、字体、图标、插画、页面组件。
- `exports/` 放之后生成的 HTML / PPT / 可画导入版，不在这里手工编辑源内容。

## 推荐工作流

1. 先看 `content/05_classroom_script.md`，确认 120 分钟课堂节奏。
2. 再改 `content/01_content_master.md`，统一课程认知和逐页内容。
3. 需要把内容讲透时，补 `content/06_story_and_cases.md`。
4. 需要图片或视频时，补 `content/07_media_plan.md`。
5. 再改 `structure/slide_manifest.yaml`，调整页面顺序和页面类型。
6. 视觉风格只改 `visual/visual_system.md`。
7. 最后再生成 HTML 或 PPT。

## 内容模块说明

- `content/01_content_master.md`：课程目标、学生产出、分段内容。
- `content/02_project_cards.md`：四个项目方向、场景题目卡、题目示范。
- `content/03_question_slides.md`：大字号课堂提问页。
- `content/04_competition_materials.md`：发明日志、查新报告和课后证据任务。
- `content/05_classroom_script.md`：120 分钟课堂节奏脚本。
- `content/06_story_and_cases.md`：故事、案例、反例、追问模板。
- `content/07_media_plan.md`：图片、视频和已有方案案例的使用位置。
