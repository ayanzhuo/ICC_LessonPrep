# 四个项目方向素材

## 冰箱食材检测

### 用户例子

- 忙碌家长：买了菜但忘记吃，最后变质。
- 老人家庭：靠经验判断新鲜度，容易不放心。
- 独居学生：食材少但管理乱，经常重复买。

### 场景题目卡

1. 忙碌家庭的临期食材提醒
2. 老人家庭的食材新鲜度判断
3. 独居学生的冰箱食材遗忘问题
4. 家庭重复购买导致的食材浪费

### 痛点示范

忙碌家庭晚饭前才发现蔬菜已经变质，既浪费钱，也影响做饭安排。

### 题目示范

为了解决忙碌家庭在日常做饭前难以及时发现临期食材的问题，我们希望设计一个能提醒临期和遗忘食材的方案。

### 根本结果

目标不是“智能冰箱”，而是少浪费、早提醒、吃得安心。

## 果园果实病状检查

### 用户例子

- 果农：巡园时间有限，早期病斑容易漏看。
- 采摘园：开放前要快速判断果实状态。
- 管理人员：需要记录不同区域的果实问题。

### 场景题目卡

1. 果农日常巡园时的病果发现
2. 采摘园开放前的果实状态检查
3. 果实病斑早期识别与记录
4. 果园管理人员的批量巡检记录

### 痛点示范

果农巡园时只能凭肉眼快速看，早期小病斑容易被漏掉，后面扩散更难处理。

### 题目示范

为了解决果农巡园时容易遗漏早期病斑的问题，我们希望设计一个能辅助拍照记录和初步提示病状的方案。

### 根本结果

目标不是“农业机器人”，而是更早发现异常、减少遗漏。

## 办公区无效照明监控

### 用户例子

- 员工：离开时忘记关灯。
- 物业：很难逐个房间检查。
- 行政：想知道哪些区域经常浪费电。

### 场景题目卡

1. 无人办公室持续开灯
2. 自然光充足但室内灯未关闭
3. 会议室使用结束后照明未关闭
4. 教室/办公区分区照明浪费

### 痛点示范

会议结束后房间没人，但灯继续开到下班后，物业很难及时发现。

### 追问示范

模糊说法：办公室经常浪费电。  
主问题：怎样帮助行政更早发现无人会议室持续开灯，并记录哪些区域经常浪费？

### 题目示范

为了解决办公室无人或光线充足时仍持续开灯的问题，我们希望设计一个能发现无效照明并提醒处理的方案。

### 根本结果

目标不是“自动关灯”，而是发现浪费、提醒处理、降低能耗。

## 车辆盲区视觉提醒

### 用户例子

- 司机：倒车或右转时看不到关键区域。
- 骑行者：靠近车辆侧后方时不容易被看到。
- 行人：停车场和小区道路里风险更高。

### 场景题目卡

1. 小区道路转弯时的行人盲区
2. 停车场倒车时的后方盲区
3. 车辆右转时的骑行者盲区
4. 变道时侧后方车辆提醒

### 痛点示范

小区转弯时司机看不到侧前方行人，等发现时距离已经很近。

### 题目示范

为了解决车辆转弯或倒车时盲区观察不足的问题，我们希望设计一个能提供视觉提醒的方案。

### 根本结果

目标不是“更多屏幕”，而是关键时刻让风险被看见。
