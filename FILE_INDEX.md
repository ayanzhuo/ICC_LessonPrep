# ICC Lesson Prep File Index

## Main Working Files

- `Lesson01/lesson01.html`: current Lesson 01 classroom HTML.
- `Lesson02/lesson02.html`: current Lesson 02 classroom HTML.
- `assets/common.css`: shared visual system, page size, navigation, logo, print styles.
- `assets/logo.svg`: shared logo.
- `templates/lesson_template.html`: copy this when creating Lesson 03 and later.

## Planning Documents

- `README.md`: current project structure and collaboration rules.
- `Course_Overview.md`: six-lesson course map.
- `Course_Production_Guide.md`: how to create and maintain later lessons.

## References

- `references/source_docs/`: original DOCX planning materials.
- `references/novelty_reports/`: previous novelty-search report drafts and final sample.
- `references/module_notes/lesson01_module_notes/`: older modular lesson notes and visual references.
- `references/ppt_assets/`: images used by older PPT export experiments.

## Tools

- `tools/legacy_export_icc_lesson1_pptx.py`: old PPT export script kept for reference. It still contains old paths and should be reviewed before reuse.

## Archive

- `_archive/legacy_lessonprep/`: previous English copy of the lesson-prep folder.
- `_archive/old_html_exports/`: old single-file HTML exports from before the current modular structure.

## Rule of Thumb

Use files in `Lesson01`, `Lesson02`, `assets`, and `templates` for current production.
Use `references` only for looking up source material.
Use `_archive` only when recovering old versions.
